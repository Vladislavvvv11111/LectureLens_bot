# LectureLens Bot

Telegram-бот для обмена учебными материалами: загрузка, поиск по тегам, оценка файлов.

## Функции
- Указать имя через кнопку
- Загрузить файл с тегами
- Поиск по ключевым словам
- Оценка файлов (1–5)
- Скачивание найденных файлов
- Личный профиль с вашими загрузками
